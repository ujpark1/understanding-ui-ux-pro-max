설계 철학: "어떤 차트가 예쁜가"가 아니라 "데이터가 어떤 질문에 답하는가"로 분류합니다. 25가지 데이터 유형마다 최적 차트 + 대안 + 쓰면 안 되는 조건 + 데이터 볼륨 임계값(SVG↔Canvas 전환 기준)을 지정합니다.

접근성 등급이 핵심 차별점: 모든 차트에 AAA~D 등급과 의무 대체 수단이 붙어 있습니다. 파이(C)·네트워크 그래프(D)·3D(D)는 "단독 사용 금지, 반드시 테이블 병기"로 명시됩니다.

실무형 커버리지: 캔들스틱(트레이딩), 이상 탐지(모니터링), 실시간 스트리밍(IoT), 프로세스 마이닝, 감성 분석 등 일반 튜토리얼에 없는 도메인 특화 유형이 절반쯤 됩니다.

25개 차트 유형 전체

카드에 붙은 접근성 등급 배지의 의미 — AAA / AA 그대로 써도 안전 · B 데이터 테이블 등 대체 수단 권장 · C / D 대체 수단 없이는 프로덕션 사용 금지. 각 카드의 그림은 해당 차트 유형의 형태 예시입니다.
