어떤 그래프를 써야 할까?

“이 데이터를 어떻게 보여주지?” 막막할 때 펼쳐 보세요. 보여주려는 내용에 딱 맞는 그래프와, 반대로 피해야 할 그래프, 그리고 누구나 보게 만드는 법까지 상황별로 정리했어요.

25상황별 그래프 추천
11누구나 읽기 쉬운 그래프
7표를 꼭 곁들여야 하는 그래프

예쁜 그래프가 아니라 ‘내 데이터에 맞는’ 그래프를 고르는 게 먼저예요. 같은 숫자라도 “시간에 따른 변화”인지 “항목 비교”인지 “전체 중 비율”인지에 따라 어울리는 그래프가 완전히 달라지거든요.

👎 ‘피하세요’ 부분을 특히 봐주세요. 예를 들어 파이 차트는 조각이 5개만 넘어도 눈으로 구분이 안 돼요. 흔히들 쓰지만 상황에 안 맞으면 오히려 헷갈리는 그래프들이 있답니다.

♿ ‘누구나 보게 하려면’도 꼭 챙겨 주세요. 우리나라 남성 20명 중 1명은 색약이에요. 색만으로 정보를 전하면 안 보이거든요.

✅ 누구나 읽기 쉬워요 🙂 무난해요 ⚠️ 표를 같이 🚫 단독 사용 금물

각 카드 맨 아래 배지는 “색약인 분이나 처음 보는 사람도 이 그래프를 이해할 수 있을까?”를 알려줘요.

상황별 그래프 25가지

라인 차트
01

시간에 따른 추세

라인 차트

시간이 지나면서 숫자가 오르락내리락하는 걸 선으로 이어 보여주는 그래프예요.

👍 이럴 때 좋아요매출·방문자 수처럼 “시간에 따라 어떻게 변했나”를 볼 때
👎 이럴 땐 피하세요점이 3~4개밖에 없거나, 선이 6개 넘게 겹쳐 복잡해질 때
♿ 누구나 보게 하려면색만으로 선을 구분하지 말고 실선·점선으로도 나눠 주세요. 색약인 분을 위해 값이 담긴 표를 함께 두면 좋아요.
✅ 누구나 읽기 쉬워요
막대 차트 (가로/세로)
02

카테고리 비교

막대 차트 (가로/세로)

항목별 크기를 막대 길이로 나란히 비교하는, 가장 익숙한 그래프예요.

👍 이럴 때 좋아요“어느 상품이 제일 많이 팔렸나”처럼 항목끼리 크기를 견줄 때 (15개 이하)
👎 이럴 땐 피하세요항목이 15개가 넘거나, 시간 흐름을 볼 때 (그땐 라인이 나아요)
♿ 누구나 보게 하려면막대마다 숫자 값을 표시하고, 표로도 내려받을 수 있게 하면 누구나 정확히 읽을 수 있어요.
✅ 누구나 읽기 쉬워요
파이 / 도넛 차트
03

부분-전체 비율

파이 / 도넛 차트

전체를 100%로 두고 각 조각이 몇 %인지 원으로 나눠 보여줘요.

👍 이럴 때 좋아요조각이 5개 이하이고 “하나가 유독 크다”를 강조하고 싶을 때
👎 이럴 땐 피하세요조각이 5개를 넘거나, 크기가 비슷해 눈으로 구분이 안 될 때, 정확한 숫자가 중요할 때
♿ 누구나 보게 하려면색에만 의존하면 색약인 분은 조각 구분이 안 돼요. 조각마다 % 숫자를 적고, 옆에 표를 꼭 함께 두세요.
⚠️ 표를 꼭 같이 보여주세요
산점도 / 버블 차트
04

상관관계 / 분포 탐색

산점도 / 버블 차트

점을 흩뿌려서 두 값이 서로 관련 있는지 살펴보는 그래프예요.

👍 이럴 때 좋아요“광고비를 늘리면 매출도 느나?”처럼 두 숫자의 관계나 튀는 값을 찾을 때
👎 이럴 땐 피하세요항목이 이름표(카테고리)일 때, 점이 20개도 안 될 때, 작은 화면 위주일 때
♿ 누구나 보게 하려면색만 쓰지 말고 동그라미·세모·네모 같은 모양으로도 그룹을 구분하고, 숫자 표를 곁들이세요.
🙂 무난해요 · 숫자 표를 곁들이면 더 좋아요
히트맵 / 코로플레스
05

히트맵 / 강도

히트맵 / 코로플레스

칸마다 색을 진하게·연하게 칠해 “어디가 뜨겁고 어디가 차가운지” 보여줘요.

👍 이럴 때 좋아요“요일×시간대별로 언제 사람이 몰리나”처럼 격자 위 밀도를 볼 때
👎 이럴 땐 피하세요칸이 20개도 안 되거나, 정확한 값을 하나씩 읽어야 할 때
♿ 누구나 보게 하려면색 위에 마우스를 올리면 실제 숫자가 뜨게 하고, 범례에 눈금을 넣어 진하기가 얼마인지 알 수 있게 하세요.
🙂 무난해요 · 숫자 표를 곁들이면 더 좋아요
코로플레스 맵 / 버블 맵
06

지리 데이터

코로플레스 맵 / 버블 맵

지역별 숫자를 실제 지도 위에 색이나 동그라미 크기로 얹어 보여줘요.

👍 이럴 때 좋아요데이터에 지역·위치가 들어 있고 “어디에 몰려 있나”가 핵심일 때
👎 이럴 땐 피하세요넓은 땅이 실제보다 과장돼 보일 때, 모바일 위주일 때
♿ 누구나 보게 하려면주요 지역엔 이름표를 달고, 지역명·값을 담은 표를 함께 제공하면 지도를 못 봐도 이해할 수 있어요.
🙂 무난해요 · 숫자 표를 곁들이면 더 좋아요
퍼널 차트 / 생키
07

퍼널 / 전환 흐름

퍼널 차트 / 생키

단계를 지날수록 사람이 얼마나 빠져나가는지 깔때기 모양으로 보여줘요.

👍 이럴 때 좋아요“방문 → 가입 → 결제”처럼 단계별로 몇 %가 남는지 볼 때
👎 이럴 땐 피하세요단계 순서가 없거나, 뒤 단계가 앞보다 커질 수 있을 때, 단계가 3개도 안 될 때
♿ 누구나 보게 하려면단계마다 몇 %가 넘어갔는지 숫자를 적고, “방문 100 → 가입 40” 식의 목록으로도 볼 수 있게 하세요.
✅ 누구나 읽기 쉬워요
게이지 / 불릿 차트
08

목표 대비 성과 (KPI)

게이지 / 불릿 차트

자동차 계기판처럼 “목표까지 얼마나 왔나”를 바늘과 눈금으로 보여줘요.

👍 이럴 때 좋아요“월 목표의 몇 %”처럼 정해진 목표 하나를 보여줄 때
👎 이럴 땐 피하세요목표 자체가 없을 때, 여러 지표를 한꺼번에 비교할 때
♿ 누구나 보게 하려면바늘 위치만이 아니라 “목표의 82%”처럼 숫자를 항상 글자로 함께 보여주세요.
✅ 누구나 읽기 쉬워요
신뢰구간 밴드 라인
09

시계열 예측

신뢰구간 밴드 라인

지난 실적에 이어 앞으로의 예상 범위를 옅은 띠로 함께 그려줘요.

👍 이럴 때 좋아요“지금까지 이랬으니 앞으로 이 정도”라는 예측과 그 오차 범위를 같이 보여줄 때
👎 이럴 땐 피하세요비교할 과거 데이터가 없을 때, 예측이 너무 안 맞을 때
♿ 누구나 보게 하려면실적과 예측을 색뿐 아니라 실선·점선으로 구분하고, 둘을 켜고 끌 수 있는 버튼을 주세요.
✅ 누구나 읽기 쉬워요
하이라이트 라인 차트
10

이상 탐지

하이라이트 라인 차트

평소 흐름 중 갑자기 튀는 지점만 빨갛게 콕 짚어주는 그래프예요.

👍 이럴 때 좋아요서버·매출 모니터링처럼 “갑자기 튄 순간”을 알아채야 할 때
👎 이럴 땐 피하세요튀는 게 미리 정해진 종류일 때, 멈춤 버튼 없는 실시간 화면일 때
♿ 누구나 보게 하려면튄 지점을 색만이 아니라 모양(동그라미 등)으로도 표시하고, “몇 시에 무슨 일”이라는 요약 목록을 옆에 두세요.
✅ 누구나 읽기 쉬워요
트리맵
11

계층 / 중첩 데이터

트리맵

큰 네모 안을 작은 네모로 쪼개 “무엇이 얼마를 차지하나”를 넓이로 보여줘요.

👍 이럴 때 좋아요“예산이 부서별로 어떻게 나뉘나”처럼 큰 덩어리 구성을 볼 때
👎 이럴 땐 피하세요3단계 넘게 깊어질 때, 비슷한 조각을 정확히 비교해야 할 때
♿ 누구나 보게 하려면색약인 분에겐 넓이 비교가 어려워요. 접었다 펴는 표를 기본으로 두고, 트리맵은 보조로만 쓰세요.
⚠️ 표를 꼭 같이 보여주세요
생키 다이어그램
12

흐름 / 프로세스 데이터

생키 다이어그램

여러 갈래로 흘러가는 양을 굵기 다른 물줄기처럼 이어 보여줘요.

👍 이럴 때 좋아요“돈·사람이 어디서 어디로 흘러갔나”처럼 출발→도착 분배를 볼 때
👎 이럴 땐 피하세요흐름이 되돌아오는 고리가 있을 때, 출발-도착 쌍이 3개도 안 될 때, 모바일 위주일 때
♿ 누구나 보게 하려면흐름을 색만으로 전하지 말고 “출발 → 도착 → 양”을 담은 표를 함께 제공하세요.
⚠️ 표를 꼭 같이 보여주세요
워터폴 차트
13

누적 변화 (증감 분해)

워터폴 차트

시작값에서 +와 −가 쌓여 최종값이 되는 과정을 계단처럼 보여줘요.

👍 이럴 때 좋아요“매출에서 이런저런 비용을 빼면 순이익”처럼 증감 과정을 볼 때
👎 이럴 땐 피하세요막대가 12개 넘게 많을 때, 청중이 그냥 최종 합계만 궁금할 때
♿ 누구나 보게 하려면오름/내림을 색뿐 아니라 ▲▼ 화살표로도 표시하고, 막대마다 숫자 라벨을 붙이세요.
✅ 누구나 읽기 쉬워요
레이더 / 스파이더 차트
14

다변수 비교

레이더 / 스파이더 차트

여러 항목 점수를 방사형 거미줄로 펼쳐 대상끼리 겹쳐 비교해요.

👍 이럴 때 좋아요“제품 A vs B를 여러 기능 점수로 한눈에” 비교할 때
👎 이럴 땐 피하세요축이 8개를 넘을 때, 정확한 수치 비교가 필요할 때 (막대가 나아요)
♿ 누구나 보게 하려면축을 5~8개로 줄이고, 정확히 비교하고 싶은 분을 위해 막대 그래프나 표를 함께 제공하세요.
🙂 무난해요 · 숫자 표를 곁들이면 더 좋아요
캔들스틱 차트
15

주식 / 트레이딩 OHLC

캔들스틱 차트

하루의 시가·종가·고가·저가를 양초 모양 하나로 압축한, 주식 전용 그래프예요.

👍 이럴 때 좋아요트레이딩·투자처럼 시가/고가/저가/종가가 다 있는 금융 데이터일 때
👎 이럴 땐 피하세요금융이 아닌 일반 청중, 시가·종가 데이터가 없을 때 (라인이 나아요)
♿ 누구나 보게 하려면상승/하락을 색만이 아니라 “속이 찬 몸통 vs 빈 몸통”으로도 구분하고, 숫자 표를 곁들이세요.
🙂 무난해요 · 숫자 표를 곁들이면 더 좋아요
네트워크 그래프
16

관계 / 연결 데이터

네트워크 그래프

점과 선으로 “누가 누구와 연결됐나”를 지도처럼 펼쳐 보여줘요.

👍 이럴 때 좋아요소셜 관계·시스템 연결처럼 개체 간 이어짐을 탐색할 때
👎 이럴 땐 피하세요점이 500개 넘게 뒤엉킬 때, 정확한 연결 수가 필요할 때, 모바일일 때
♿ 누구나 보게 하려면이 그래프는 화면 읽어주는 프로그램으론 이해가 거의 불가능해요. “A—B 연결” 목록 표를 반드시 함께 두세요.
🚫 단독 사용은 피하세요 · 표 필수

박스 플롯

👍 이럴 때 좋아요“그룹별로 성적이 어떻게 퍼졌나”처럼 분포·중앙값·튀는 값을 비교할 때
👎 이럴 땐 피하세요그룹당 데이터가 20개도 안 될 때, 통계 그래프가 낯선 청중일 때
♿ 누구나 보게 하려면최소·중앙·최대 같은 값을 담은 요약 표를 함께 두고, 튀는 값이 몇 개인지 글로 적어 주세요.
✅ 누구나 읽기 쉬워요
불릿 차트
18

목표 대비 성과 (컴팩트)

불릿 차트

게이지를 납작하게 눕힌 버전으로, 목표선 대비 현재를 막대 하나로 보여줘요.

👍 이럴 때 좋아요대시보드에 여러 KPI를 좁은 공간에 나란히 놓을 때
